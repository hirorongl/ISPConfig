ispCPというコントロールパネルを長年使ってきたがプロジェクトが長らくメンテナンスされていないため代替えのコントロールパネルとしてISPConfigを利用してみる。

プロジェクト本家URL： http://www.ispconfig.org/


テスト使用してみて、日本語の言語ファイルに文字化けが存在したのでそれを修正したものをここで公開。

  \interface\web\dns\lib\lang

上記フォルダに修正したものを格納


ついでに簡単に管理画面の言語エディタのインポートから取り込めるようにISPConfig3.0.5.2用のエクスポート言語ファイル(export_3.0.5.2_ja.lng)を置いておく。
